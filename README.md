# Ubuntu initial setting files
There are two files for initial setting of Ubuntu.
If you install the Ubuntu to your computer, you will be able to run these files which is written in shell script.

- setup1.sh
This file can install vim, git, curl, ssh, and settings for Japanese-input.
This file contains my `.vimrc` that is very usefull in my opinion.
I guess almost all users are using the VSCode for coding, so they don't need it.
However, I shared it just in case when I have to fix some trable.
You can rewrite it if you want to use yours, of course.

This Japanese-input setting is only for the US-keyboard.
That's why I don't know how to refine it for the JIS-keyboard.
I hope that you can find nice documents.
If you cannot do anymore, I will do my best:D

- setup2.sh
You can install Google chrome, Visual Studio Code, gcc with lapack and blus, and Anaconda from this installer.
This Anaconda installer was available at March 16, 2026.
If you want to install another one, you can delete these lines.
